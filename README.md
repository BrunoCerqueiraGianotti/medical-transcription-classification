# 🏥 Análise e Classificação de Transcrições Médicas

Projeto acadêmico desenvolvido para a disciplina de Projeto Aplicado II, com foco na aplicação de técnicas de Ciência de Dados e Processamento de Linguagem Natural (NLP) na área da saúde.

## 📌 Objetivo

Desenvolver um modelo de Machine Learning capaz de classificar automaticamente transcrições médicas de acordo com sua especialidade clínica.

## 📊 Dataset

O projeto utiliza o dataset **Medical Transcriptions**, disponível no Kaggle:

https://www.kaggle.com/datasets/tboyle10/medicaltranscriptions

- Aproximadamente 5.000 registros
- Dados textuais (não estruturados)
- Especialidades médicas diversas

- [x] Definição do problema
